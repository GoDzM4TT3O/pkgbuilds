# PKGBUILDs

[This repository](https://github.com/GoDzM4TT3O/pkgbuilds) contains PKGBUILDS, used on Arch-based distros to install software.

For more PKGBUILDs, visit [my website](https://godzm4tt3o.js.org/aur.html).
